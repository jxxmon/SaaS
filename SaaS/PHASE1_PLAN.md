# Phase 1: Next.js Hub Container 개발 (@Port 3000)

## 목표
- Next.js Hub 기본 구조 구현
- 다국어 라우팅 완성
- 도구 카테고리 페이지 구현
- 개별 도구 상세 페이지 구현
- SEO 최적화 적용

## 구현 항목

### 1. 레이아웃 컴포넌트
-  - 메인 레이아웃
- 헤더 컴포넌트 (언어 전환, 검색, 네비게이션)
- 푸터 컴포넌트
- AdSense Placeholder 영역 구현

### 2. 메인 페이지
-  - 홈페이지
- Featured 도구 카드 그리드
- 카테고리별 도구 섹션
- 검색 기능 프리뷰

### 3. 도구 라우팅
-  - 개별 도구 페이지
- 도구 상세 정보 표시
- 도구 실행 인터페이스 (클라이언트/Worker 분기)
- 관련 도구 추천

### 4. API 라우트
-  - 도구 데이터 API
-  - Worker 프록시 API

### 5. 공통 컴포넌트
- 도구 카드 컴포넌트
- 검색 컴포넌트
- 언어 전환 컴포넌트
- 로딩/에러 컴포넌트

## 개발 순서
1. 기본 레이아웃 및 컴포넌트 구현
2. 메인 페이지 구현
3. 도구 상세 페이지 구현
4. API 라우트 구현
5. SEO 최적화 적용
